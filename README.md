# HH.ru Data Project – Собранный и проанализированный датасет Вакансий

Проект демонстрирует сбор, хранение и анализ вакансий с hh.ru с последующей визуализацией данных.  


## Основные этапы проекта

1. Загружал вакансии через API hh.ru и сохранял в облачное raw-хранилище (S3) в **Yandex Cloud**:
    - Собирал джейсоны с помощью **request**
    - Для подключения к Яндексу использовал **boto3**

2. Загружал данные из S3 staging таблицу в PostgreSQL в виде id, json, date. Затем переносил, доставая информацию из JSON в clean таблицу. Все с помощью **python** скриптов. Для взаимодействия с PostgreSQL использовал:
    - **sqlalchemy**
    - **psycopg2**

3. Проводил анализ данных, рассматривая их в **SQL**. Также проводил анализ текста с помощью sklearn, проверяя наиболее часто встречающиеся слова и фразы. Визуализировал с помощью WordCloud 

4. Визуализировал данные в **METABASE**, предварительно запустив контейнер с ним. Дашборд лежит в директории **dashboard**


## Основные выводы анализа

- Рынок Data-вакансий сильно сосредоточен в Москве  
- Средняя зарплата: от 167 000 ₽ до 198 000 ₽  
- Формат работы: примерно поровну между удалёнкой и офисом  
- Большинство вакансий рассчитано на специалистов с опытом 1–3 года или 3-6 лет  
- Зафиксированны пики публикаций, которые могут отражать внутренние циклы компаний
